# 玩了會懷疑人生系列-1-貪食蛇(Linux版)
BUILD WITH CODEBLOCKS RUN IN DEBAIN

1. 執行下列指令
```
 sudo apt-get install git libncurses5-dev g++ -y
 git clone https://github.com/chenlicpp/fen-game-1-linux.git
 cd fen-game-1-linux
 g++ -lcurses -o snake main.cpp fn.cpp
 ./snake
```
